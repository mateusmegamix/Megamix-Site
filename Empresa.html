O que é

O Megamix é uma uma empresa na qual é responsável em fazer sistemas próprios para usuários que estejam começando seu próprio negócio. Tendo em vista a competência e profissionalidade no desenvolvimento de acordo com os requisitos do cliente. Colocamos em prática as tecnologias mais atuais do mercado e oferemos total suporte aos sistemas desenvolvidos. Trabalhamos também com montagem de computadores do mais simples ao mais abrangentes. Prestamos também serviços de manutenção e Instalação de Softwares.


Tecnologias Utilizadas

Utilizamos Android Studio para aplicativos android, Visual Studio 2019 para desenvolver sistemas desktops e HTML, CSS e JavaScript para criação de sites. Com banco de dados Sybase e SqlServer para armazenamento de dados. Em sistemas desktop é utilizado Crystal Report para desenvolvimento de relatórios. Trabalhamos com diversas linguagens de Programação (PHP, Java, C#, JavaScript). Para Suporte Técnico utilizamos Tean Viewer como acesso remoto ao computador do cliente.


Como funciona nosso trabalho

Megamix está situado na cidade de Resende, um município do sul do estado do Rio de Janeiro(região Sul Fluminense), que faz divisa entre os estados de São Paulo e Minas Gerais. Nossa equipe conta com 2 desnvolvedores Analista de Sistemas. Trabalhamos com objetivo nas funções designadas, conforme o estabelecido por requisitos. O Cliente entra em contato para agendar uma reunião, após reunião é feito um orçamento.

Desenvolvimento

Desenvolver um sistema de gestão próprio é encarar uma jornada longa, com várias etapas e muitos imprevistos. Vale considerar que construir um software de gestão do zero e colocá-lo em funcionamento vai de acordo com a quantidade de requisitos. Além disso, há sempre um período de testes, pois é natural que o sistema recém-criado apresente inconsistências; Para isso estaremos sempre matendo contato com o cliente, para fazermos as atualizações necessárias.
