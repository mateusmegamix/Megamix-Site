Montagem e Manutenção

Computadores e Notebooks

Manutenção de máquinas, geralmente, não é algo com o qual nos preocupamos, ela simplesmente acontece. São raros os casos de usuários que tomam o cuidado de aderir a manutenção preventiva periódica, que são muito importantes. Para isso trabalhamos com consertos de computadores e notebooks.

Serviços




• Formatação com backup de seus arquivos
• Upgrade de peças e acessórios
• Manutenção preventiva e corretiva
• Limpeza interna dos componentes
• Instalação de periféricos
• Instalação de programas
• Retirada de virus e spyware

Desenvolvimento

Tecnologias Utilizadas

Utilizamos Android Studio para aplicativos android, Visual Studio 2019 para desenvolver sistemas desktops e HTML, CSS e JavaScript para criação de sites. Com banco de dados Sybase e SqlServer para armazenamento de dados. Em sistemas desktop é utilizado Crystal Report para desenvolvimento de relatórios. Trabalhamos com diversas linguagens de Programação (PHP, Java, C#, JavaScript). Para Suporte Técnico utilizamos Tean Viewer como acesso remoto ao computador do cliente.
